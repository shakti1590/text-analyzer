<h3>
Text Analyzer
</h3>

It has the following features:

- Convert Text to UpperCase
- Convert Text to LowerCase
- Convert Text to SentenceCase
- Encode to base64
- Clear Text
- Extract Numbers
- Extract Links
- Listen text and stop it
- Remove white space
- Remove special characters
- Copy to clipboard
- Reverse text
- Change text
- Undo action
- Redo action
- Preview input text
- Count characters, words, statements, questions

#### Technology Used:
 - React Js
 - Java Script
 - HTML
 - CSS
